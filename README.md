# ⚽ Football Data Analysis

## 📊 Overview
This project analyzes football match data to identify team performance and match insights.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI

## 📈 Key Insights
- Top teams perform consistently
- Goal scoring impacts winning chances
- Match trends identified

## 🚀 Outcome
Built data-driven insights and dashboards for football analytics.
